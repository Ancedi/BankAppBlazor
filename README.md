# BankAppBlazor

#Körinstruktioner
1. Alla sidor utom Home(inloggnings sidan) är ej tillgängliga tills inloggning sker. För att logga in, skriv "Admin" hos username kolumnen och "1234" hos password kolumnen i Home.
Efter inloggning, kan du skapa och radera flera bank konton, eller logga ut med "Log Out" knappen under rubriken.
2. För att skapa ett konto, fyll i "Create Account" formen i Create Account sidan. Därefter visas en lista av tillgängliga/existerande konton.
3. För att radera ett konto, klicka på "Delete Account" knappen brevid de individa konton som visas i listan.
4. Lisan visar alla konton konto typ, balans, och när sist de interagerades med (skapades, överförde eller fick "pengar"). För att uppdatera balans efter ränta, tryck "Check Interest".
5. För att överföra pengar mellan konton, gå till "Transfer" och välj kontot som överför pengar överst och vilken som får under. Nedest fills in balans som överförs.
6. "Transaction History" sidan visar alla överföringar som de respektiva konton har genomgått. Användaren kan klicka på pricken till vänster av "Sort By Date" eller "Sort By Amount" för sortera genom tidigast till senaste överföringar, eller från dyraste till billigaste. Till höger av dem visar en pil om de går från högst till lägst eller vice versa. Ovanför finns en kolumn som användaren kan använda för att fitrera bort överföringar mindre än en vis mängd från att visas i listan vald av användaren.
